= github auth status
View authentication status.

== NAME
github auth status

== SYNOPSIS
====
[source]
----
github auth status --show-token boolean --help 
----
====

== OPTIONS
`--show-token` boolean Display the auth token. *Optional, default = false*

`--help or -h`  help for github auth status *Optional*
