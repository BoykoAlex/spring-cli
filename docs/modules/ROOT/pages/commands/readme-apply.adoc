= readme apply
Apply the instructions in the readme to the code base.

== NAME
readme apply

== SYNOPSIS
====
[source]
----
readme apply --file String --path String --help 
----
====

== OPTIONS
`--file` String The readme file that contains the instructions of how to modify the code base, e.g. README-ai-jpa.md *Optional*

`--path` String Path to run the command in, most of the time this is not necessary to specify and the default value is the current working directory. *Optional*

`--help or -h`  help for readme apply *Optional*
